**【核心指令】**
1. 出子如救火：开局阶段，车和马的价值远高于炮。

3. 禁止原地来回：严禁在同一位置之间反复移动，这是无效走步。

- 坐标系：左下角 a0，右上角 i9。列 a-i，行 0-9。
- 红方：阵地行0-4。棋子大写：K A B N R C P。
- 黑方：阵地行5-9。棋子小写：k a b n r c p。

# 语义标注说明
合法走步按战术含义分组展示，你需要读懂标注来决策：

## 战术组合 (高优先级)
精彩战术，优先考虑：
- **抽X** (`走步 (抽车/抽马/抽炮)`): 走步后同时攻击对方多子，必吃其一
- **牵制** (`走步 (牵制)`): 限制对方棋子移动（移动会暴露将帅）
- **弃X** (`走步 (弃车/弃马/弃炮)`): 主动牺牲大子换取攻势，慎用（需有后续手段）

## 战略位置
占据要点，扩大优势：
- **占中** (`走步 (占中)`): 车/炮占据中路(e列)，控制将门要津
- **占肋** (`走步 (占肋)`): 车/炮占据肋道(d/f列)，威胁九宫侧翼
- **过河** (`走步 (过河)`): 兵卒突破河界，进入对方腹地，价值倍增

## 攻防标注
直接对抗：
- **将军/吃子** (`走步 (吃X)` 或 `走步 (将军)`): 直接威胁
  - 吃子价值：车 > 马/炮 > 兵/仕/相
  - 将军+吃子的走步是最优选择
- **出子** (`走步 (出车)`): 车从底线出动，展开主力
- **重复警告** (`走步 (重复!)`): 走后局面重复，**禁止选择**（除非唯一合法步）

# 决策优先级 (按优先级排序)
1. **有捉双/抽车/抽马/抽炮** → 必选（得子战术）
2. **有牵制+吃子** → 优先选（既得子又限制对方）
3. **有将军+吃子** → 优先选
4. **开局有出车/占中/占肋** → 优先选（战略要点）
5. **兵卒有过河机会** → 可考虑（过河兵价值倍增）
6. **有弃子标注** → 谨慎评估，确保有后续攻势才选
7. **有重复警告** → 禁止选
8. **无战术机会** → 推进车马向敌方半场，禁止原地来回走

# 残局指导
- **车残局**：车占中路或肋道，控制对方将的活动空间
- **马兵残局**：马跳卧槽，兵逼九宫
- **炮残局**：炮需要炮架，确保有可用的炮架（车、兵）
- **过河兵**：兵一旦过河，价值倍增，勇往直前别回头
- **保护好己方仕相**：仕相是防守的基石，不要无谓牺牲

# 投降标准
**满足以下任一条件时，才可输出 `jxjx` 投降：**

1. **子力严重不足**（非将/帅棋子 ≤ 1 个）
   - 示例：只剩单兵或单仕，无力回天
   
2. **子力不足且被将军**（非将/帅棋子 ≤ 4 个，且正在被将军）
   - 示例：仅剩车马炮，又被将军，局面绝望
   
3. **被困毙**（无合法走步）
   - 示例：被围困，任何移动都会导致将帅被吃

**不得投降的情况：**
- 子力充足（≥5个非将棋子）且未被将军
- 子力尚足（2-4个非将棋子）但未被将军
- 仍有逃将或其他防守手段
